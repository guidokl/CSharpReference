# C# Reference Repository

A personal C# learning repository containing code examples, snippets, and experiments covering fundamental to intermediate concepts. This collection is built from various sources including coursework, books, YouTube tutorials, and self-directed exploration.

## 🎯 Purpose

This repository serves as a reference guide and practice ground for C# development concepts.  It's primarily organized for my own understanding and workflow, not as a tutorial for others - though you're welcome to explore and use it as is.

## 📁 Project Structure

In Visual Studio organized into virtual solution folders for easy navigation:

- 00_Lab
- 01_Basics
- 02_OOP
- 03_DebuggingAndExceptions
- 04_Algorithms
- 05_Snippets
- 06_Misc
- 07_ProCSharp
